Electronics design

In this part, we will be talking about electronics design, all about the designing of PCBs.

Introduction

Before we mill out the PCB, we need to design the board by finding the right components and placements to create the circuit board wanted.

software for PCB designing

There are some software we can use for designing PCBs, some of the examples include:

  • Autodesk EAGLES.
  • Kicad.
  • EasyEDA.
  • Fusion 360 CAE.
  • Fritzing.

We will be using EAGLES for this elective.

Sequence of designing

The sequence of designing the PCBs is as follows:

  1. Schematic entry.
  2. Board layout.
  3. Routing.
  4. Simulation.
  5. Fabrication

In these software you can download libraries, libraries are files that contain different components that can be used in the software for designing, EAGLES have a pre installed library for basic components, and we can always add more if needed.
